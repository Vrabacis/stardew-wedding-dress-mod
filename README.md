# Wedding Dress Mod для Stardew Valley

Мод добавляет красивое свадебное платье в игру.

## Установка

1. Убедитесь, что установлен **SMAPI** (https://smapi.io/)
2. Скачайте этот мод
3. Распакуйте папку в папку `Mods` Stardew Valley
4. Запустите игру через SMAPI

## Как получить платье в игре

Свадебное платье можно купить у портного или получить через игровые события.

## Требования

- SMAPI 3.13.0+
- Stardew Valley 1.5+
- Content Patcher
