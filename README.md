# Wedding Dress Mod для Stardew Valley

Мод добавляет красивое свадебное платье в игру Stardew Valley.

## Особенности

✨ Свадебное платье в оригинальном пиксель-арт стиле
✨ Появляется в магазине портного (Desney's Tailoring Shop)
✨ Работает для всех направлений движения (вверх, вниз, влево, вправо)
✨ Поддерживает анимацию ходьбы

## Установка

### Требования:
- **SMAPI 3.13.0+** (https://smapi.io/)
- **Content Patcher** (https://www.nexusmods.com/stardewvalley/mods/1915)
- Stardew Valley 1.5+

### Шаги установки:

1. Установите SMAPI, если еще не установлен
2. Установите Content Patcher мод
3. Скачайте этот мод
4. Распакуйте папку `stardew-wedding-dress-mod` в папку `Mods` вашей игры:
   - Windows: `%appdata%\StardewValley\Mods`
   - macOS/Linux: `~/.config/StardewValley/Mods`
5. Запустите игру через SMAPI

## Как получить платье в игре

1. Перейдите в магазин портного (Desney's Tailoring Shop)
2. Купите "Wedding Dress" за 5000 золота
3. Наденьте платье!

## Технические детали

- **Размер спрайта:** Оригинальный размер Stardew Valley
- **Направления:** 4 (вверх, вниз, влево, вправо)
- **Анимация:** Поддерживается при ходьбе
- **Совместимость:** Работает как для персонажей-женщин, так и для мужчин
